## 📱 Screens

### 🔹 WelcomeScreen
* Intro UI before login

### 🔹 LoginScreen
* Email & password form  
* Input validation for empty fields

### 🔹 RegisterScreen
* Registration form with validation  
* CustomTextFormField usage

### 🔹 VerifyMobileScreen
* Enter mobile number for OTP  
* Form validation implemented

### 🔹 VerifyCodeScreen
* OTP input using **Pinput**  
* Validation and onCompleted callback  
* Navigation to next screen

### 🔹 HomeScreen
* Main content feed UI  
* Shows products and categories

### 🔹 FollowHomeScreen
* Followed stores or items feed  
* Displays updates from followed entities

### 🔹 SearchScreen
* Search bar and results  
* Filters and suggestions UI

### 🔹 FilteredSearchScreen
* Displays search results based on filters  
* Shows product list with sorting and search suggestions

### 🔹 ExploreScreen
* Explore new products, categories, or deals  
* Interactive UI for discovery

### 🔹 FruitsScreen
* Displays a list/grid of fruit products  
* Interactive UI for selecting and viewing items

### 🔹 DetailScreen
* Product detail page  
* Shows image, description, price, and Add to Cart button

### 🔹 FavouriteScreen
* List of favourited products  
* Easy navigation to product details

### 🔹 MyCartScreen
* Shows selected items in cart  
* Checkout button triggers **CheckoutBottomSheet**

### 🔹 CheckoutBottomSheet (Sheet Modal)
* Slide-up modal from cart screen  
* Shows Delivery, Payment, Promo, and Total cost rows  
* Place Order button navigates to **OrderAcceptedScreen**

### 🔹 OrderAcceptedScreen
* Order confirmation screen with SVG illustration  
* "Go To Home" button navigates back to main app with BottomNavigationBar  

### 🔹 LogOutScreen
* Confirms user logout  
* Option to navigate back to Welcome/Login screens
